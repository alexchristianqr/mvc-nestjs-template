# MVC Template NestJS

## Description

API mvc description

## Link app deployment

http://localhost:3000/

## ENV

Crea el archivo .env a partir del archivo .env.example

## Installation

```bash
$ npm install
```

## Running the app

```bash
# execute with docker
$ docker-compose up -d --build

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Set branch master to template

```bash
# ejecutar en la terminal
$ sh setbranch.sh
```
